What is "synchro_simple"?

Synchro_simple is a very simple prototype application which purpose is to synchronize files and directories for backup/archive purpose.
It was written within the OCP project.

Synchro_simple is a Qt GUI for Windows OS. The user can specifiy a source directory, where the content is taken from, and a destination directory, where content is to backup.

Because synchro_simple does not use driver, the source directory is regularely checked for modification and only the modification are synchronized.

Synchro_simple can be run in background mode and has a setting menu that allows the user to confugure it.

How to install

The easiest way is to download synchro_simple_setup.exe from http://ocpforum.org/synchro_simple/synchro_simple_setup.exe and simply run it on a Windows OS (32 or 64bits).

You can also compile the source code. It is located at https://github.com/jlguenego/sandbox.
1) Clone the repository with the given URL.
Then you will need:

  • TRE installed in /usr/local/include/tre and library in /usr/local/lib (http://laurikari.net/tre)
  • MakeNSIS, from NSIS project (http://nsis.sourceforge.net)
  • environment variables:
    • QT_MINGW_BIN : directory where Qt MinGW binaries are installed (ex: C:/Qt/Qt5.2.0/5.2.0/mingw48_32/bin).
    • QT_TOOLS_BIN : directory where Qt Tools binaries are installed (ex: C:/Qt/Qt5.2.0/Tools/mingw48_32/bin).

2) Open a MinGW console and go inside the cloned repository.
3) Go to qt/synchro_simple and run build_windows.sh
